📊 상품·할인·다품목·월별행사 관리

모든 상품과 할인 정보는 Google Sheets에서 직접 관리합니다.
시트에서 수정하면 사이트에 즉시 반영됩니다.

📊 Google Sheets 열기

📦 시트별 역할

상품 — 판매할 가전제품 추가/수정/삭제 (활성 컬럼을 N으로 하면 사이트에서 숨겨짐)
할인 — 카드, IPTV, 캐시백, 특별할인 항목 관리
다품목할인 — LG/삼성 다품목 할인액 조정
월별행사 — 매월 행사 모델/혜택 변경 (적용월 컬럼 변경)
신청 — 고객 상담 신청이 자동으로 누적되는 시트 (수정 X)

💡 주의사항

시트의 헤더 행(1행)은 절대 변경하지 마세요. 변경 시 사이트가 작동하지 않습니다.
상품 ID는 고유한 숫자여야 합니다 (중복 X).
상품을 임시로 숨기려면 '활성' 컬럼을 N으로 변경하세요.
변경 후 사이트에 안 보이면 새로고침(F5) 해주세요.

🚀 일상 관리 흐름

아침: 관리자 페이지 열어 [신규] 신청 확인
고객 응대: 상세보기 → 전화 걸기 → 상태 [상담중]으로 변경
계약 성사: 상태 [계약완료]로 변경
월말: CSV 다운로드 → 매출 정리

📦 상품 추가하는 법

1. [데이터 관리] 탭 → Google Sheets 열기
2. '상품' 시트에서 마지막 행 아래에 새 행 추가
3. ID는 기존 ID + 1로 (중복 X), 활성은 'Y'
4. 저장 → 사이트 새로고침 → 즉시 반영

🎁 월별 행사 변경하는 법

1. '월별행사' 시트의 [적용월] 컬럼을 이번달로 (예: 2026-04 → 2026-05)
2. 새 행사 모델 / 할인액 입력
3. 활성 'Y' → 사이트 자동 반영

💳 카카오톡 알림 설정 (필요시)

고객 신청이 들어오면 즉시 카카오톡으로 알림 받기 위해 카카오워크 무료 채널 설정이 필요합니다.
설정 방법은 별도 안내 드릴게요. 카카오워크 앱 설치 후 Webhook URL을 받아 Apps Script에 입력하면 즉시 알림이 옵니다.

❓ 문제 해결

사이트에 상품이 안 보여요 → '활성' 컬럼이 'Y'인지 확인
가격이 이상해요 → 시트의 숫자 셀에 쉼표(,) 없는지 확인
신청이 안 와요 → Apps Script 배포가 '모든 사용자' 권한인지 확인
